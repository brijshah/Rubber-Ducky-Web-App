# Rubber Ducky Web Application

A Web application for use with the USB Rubber Ducky

### Versions
- Node 5.30
- AngularJS 1.5.0
- Express 4.13.1

### Installation

```sh
$ npm install
```


### Build & Development
For building:
```sh
$ grunt
```
For preview:
```sh
$ grunt serve
```

### Testing
To run tests with Karma:
```sh
$ grunt test
```
